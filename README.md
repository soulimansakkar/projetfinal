# Wakdo - Projet étudiant

Petit projet de commande en ligne avec deux parties :

- `bloc1-frontend` : interface client en HTML/CSS/JS
- `bloc2-backend` : administration PHP avec login et gestion des commandes
- `bloc3-symfony` : dossier non utilisé
- `bloc5-devops` : fichiers Docker

## Installation rapide

1. Ouvrir `bloc2-backend` avec un serveur PHP / Apache.
2. Charger `bloc1-frontend/index.html` depuis un serveur local ou un serveur web.
3. Se connecter dans `bloc2-backend` pour voir les commandes.

## Notes

- le frontend charge les produits depuis `data/produits.json`
- l'API de commande est dans `bloc2-backend/api.php`
- le projet est volontairement simple, style débutant
